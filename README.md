# MarketMind AI
Autonomous Brand Intelligence & Social Media Growth Engine

## Problem
Small businesses struggle with consistent, brand-aligned social media marketing.

## Solution
MarketMind AI analyzes a business website to understand brand identity and
automatically generates, schedules, and optimizes marketing content using AI.

## Features
- Website-based brand intelligence extraction
- AI-generated captions, creatives, and short videos
- Social media scheduling & automation
- Analytics-driven optimization

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- AI: NLP, Generative AI
- APIs: Instagram, LinkedIn, Facebook, X
